# Todo App

Simple Todo Progressive web application stores your data locally and can be installed on cross platforms. 


## Tech Stack

**Client:** React, TailwindCSS

**Server:** No Server Required


## Features

- Simple and clean UI
- Stores Data locally 
- Cross platform (PWA)
- Dark/Light Mode (Upcoming)
